🚀 EASIEST METHOD: RENDER (RECOMMENDED)

STEP 1: UPLOAD TO GITHUB (2 minutes)
────────────────────────────────────
1. Open browser → Go to: https://github.com/signup
2. Create account (or login if you have one)
3. Click "+" (top right) → "New repository"
4. Repository name: shield-safety-app
5. Click "Create repository"
6. Click "uploading an existing file"
7. Select ALL files from your SHIELD folder
8. Drag and drop them into GitHub
9. Click "Commit changes"

✅ Done! Your code is on GitHub!

STEP 2: DEPLOY ON RENDER (3 minutes)
────────────────────────────────────
1. Open browser → Go to: https://render.com
2. Click "Get Started for Free"
3. Click "Sign up with GitHub"
4. Authorize Render
5. Click "New +" → "Web Service"
6. Click "Connect account" → Select GitHub
7. Find and select: shield-safety-app
8. Fill in these settings:

   Name: shield-backend
   Environment: Node
   Build Command: cd backend && npm install
   Start Command: cd backend && npm start
   
9. Click "Create Web Service"
10. Wait 2-3 minutes (watch the logs)

✅ Done! Your backend is live!

You'll get a URL like: https://shield-backend.onrender.com

STEP 3: UPDATE FRONTEND (1 minute)
────────────────────────────────────
1. Open: frontend/js/auth.js
2. Find line 6: const API_BASE_URL = 'http://localhost:3000/api';
3. Change to: const API_BASE_URL = 'https://shield-backend.onrender.com/api';
   (use YOUR actual Render URL)
4. Save the file
5. Go back to GitHub
6. Upload the updated auth.js file
7. Render will auto-redeploy

✅ Done! Your app is fully deployed!

STEP 4: TEST YOUR DEPLOYMENT
────────────────────────────────────
1. Visit: https://shield-backend.onrender.com
2. Click "Sign Up"
3. Create an account
4. Login
5. Test SOS button
6. Test AI chatbot

✅ Everything working? Congratulations! 🎉
